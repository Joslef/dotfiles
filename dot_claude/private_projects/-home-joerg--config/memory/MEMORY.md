# Memory — ~/.config

## Hyprland (`~/.config/hypr/`)
- Main config: `hyprland.conf` (autogenerated base, user-edited)
- Split configs: `monitors.conf`, `workspaces.conf`
- Lock/wallpaper: `hyprlock.conf`, `hyprpaper.conf`
- Scripts: `opacity-control.sh`, `random-wall`
- Reload: `hyprctl reload`

## Waybar (`~/.config/waybar/`)
- Config: `config.jsonc`, Style: `style.css`
- Version: v0.15.0
- Scripts: `updates.sh`, `weather.sh`
- **mpris module**: `{dynamic}` and `dynamic-order` NOT supported in v0.15.0 — use `{title}`, `{artist}`, `{album}` only. Unsupported options cause crash on new MPRIS player connect.
- mpv radio via `mpv-mpris` package. mpv icon: `󰐻`, player key: `"mpv"`.
- Reload: `pkill -SIGUSR2 waybar` (must run in compositor session, not Claude shell).

## Neovim (`~/.config/nvim/`)
- Framework: LazyVim (lazy.nvim bootstrap in `init.lua`)
- Config structure: `lua/config/` (autocmds, keymaps, lazy, options)
- Plugins: `lua/plugins/` — claudecode, colorscheme, lazygit, lualine, opencode, windsurf, yazi
- Style: `stylua.toml` for Lua formatting

## pkgsync (`~/.config/pkgsync/`)
- Single config file pointing to repo at `~/Documents/pkglist`
- Remote: GitHub (`Joslef/CachyOS_pkglist`)

## System
- OS: CachyOS (Arch-based), shell: fish
- WM: Hyprland
- `playerctl` installed at `/usr/bin/playerctl`
- No `socat` or `nc`; Python3 socket available as IPC fallback
