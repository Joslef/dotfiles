- Terminal: **kitty** (primary) + **ghostty** (also used)
- File manager: **Dolphin** (WM class: `org.kde.dolphin`)
- Scratchpad terminal bound to Super+` (`special:term` workspace)
- Nerd Font in use: **JetBrainsMono Nerd Font** (v3, supports supplementary plane icons)
